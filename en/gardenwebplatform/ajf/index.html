Install Apache jena fuseki

Download Apache Jena Fuseki from the official website.

  • Official site

Unzip the downloaded zip file to any location.

The unzipped Apache Jena Fuseki will be placed.

Double-click fuseki-server.bat to execute it.

When executed, the command prompt will be launched.

Visit Apache Jena Fuseki.

  • localhost:3030

Please do the following:

  1. Select “manage datasets”
  2. select “add new dataset”
  3. Enter “garden” for the Dataset name
  4. Select “Persistent ( TDB2 )” of Dataset type
  5. Press create dataset

A dataset is created.

That’s all for building the environment of Apache Jena Fuseki.
