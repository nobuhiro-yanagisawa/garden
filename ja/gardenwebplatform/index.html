新規に導入する場合

Garden Webplatformを新規に導入する場合の手順は以下の通りです。

  1. JAVA HOMEを設定する

  2. Tomcatをインストールする

  3. PostgreSQLをインストールする

  4. Apache Jena Fusekiをインストールする

  5. ジョブを起動する

  6. アプリを起動する
